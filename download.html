Download GalliumOS

GalliumOS can be installed by downloading an ISO below or by using the chx script from ChromeOS. If you wish to dual boot, chrx is the easiest method. If you want to wipe ChromeOS and have a complete GalliumOS system, download the ISO below.

See the wiki for instructions on how to install GalliumOS with chrx or by installing the ISO.

Please check out the hardware compatibility list to see if your device is supported, which build to download, and any issues reported for your device.
